# SpriteBrew 🧪

AI-powered pixel art sprite sheet generator. Upload or describe a character → generate animation sprite sheets → preview with keyboard controls → export game-ready sheets.

**Live at [spritebrew.com](https://spritebrew.com)**

## What It Does

- **Upload & Slice** — Drop a sprite sheet PNG, define frame dimensions, auto-detect grid, slice into individual frames
- **Preview & Play** — Test animations in a live sandbox with keyboard controls (arrow keys, spacebar, etc.)
- **Export Anywhere** — Unity, Godot, GameMaker, RPG Maker, Aseprite — one click
- **AI Generation** (Coming Soon) — Describe a character or upload a reference → generate full animation sprite sheets

## Tech Stack

Next.js 14+ · React 19 · TypeScript · Tailwind CSS · PixiJS · Zustand
